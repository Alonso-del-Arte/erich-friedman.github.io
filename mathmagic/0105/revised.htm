Ambiguous Products

C.Baiocchi

Abstract

We want to study a particular case of the problem posed by Erich Friedman on January 2005 in http://www.stetson.edu/~efriedma/mathmagic/. More precisely, we seek for three numerical strings A, B, C such that:
A×BC = AB×C
(1)
where a string like XY stays for the concatenation of the strings X and Y.

The first section contains some statements about the triples (A,B,C) satisfying (1). The second section is devoted to proofs. In the last section we discuss the problem of arbitrarily long solutions.

1  Statements

Theorem 1 Let (A,B,C) be any solution of (1). Then:

  • r1 > 0; r1 simplifies to an integral value;
  • r2 £ 1; if r2 < 1 then r2 is a periodic number without nonperiodic part;
  • 1/10 £ r3 < 1; r3 has a finite decimal expansion.

Theorem 6 If (A,B,C) solves (1), then (AB,B,BC) is also a solution.

3  Arbitrarily Large Solutions

We now discuss another problem posed by Erich:

are there arbitrarily large solutions of (1)?

A positive answer is given by the following three classes of solutions:

  • fix any solution (A,B,C) and replace C by C[0]k;
  • fix any h,k integral, and any D; then define A = C = [D]h, B = [D]k;
  • fix any solution (A,B,C) and replace A by A[B]j, B by [B]k and C by [B]jC.

Many less trivial examples can be constructed by using (B) and/or (C). On the other hand, a very general result in this direction is due to Emilio Schiavi; see again the "solution page" of Friedman at http://www.stetson.edu/ efriedma/mathmagic/0105.html. With the notations (2), it refers to the limit case of r2 = 1, say B = 9 or, more generally, B = 99¼9. Then property (C1) holds true, because the fraction r3 becomes A/(A+1). Thus the necessary and sufficient condition reads:

A+1 has only 2 and/or 5 as prime factors.
In other words:

  • Fix any ``denominator'' D of the form D = 2m×5n (m,n ³ 0, n+m > 0);
  • choose as A any factor of D-1;
  • set C = A×(10j)/( A+1)   (j ³ max(m,n));
  • fix any k > 0 and set B = 10k-1.

Then the triple (A,B,C) solves (1); better, these triples describe all the solutions with r2 = 1.

Problem of the Month (July 2019)

This month we consider several problems involving products, quotients, and divisors.

Problem #1:

There are no numbers of more than one digit that are the product of their own digits. But if we are willing to multiply by one more digit, there are some. And if we are willing to repeat the process, multiplying by possibly different numbers in order to obtain the original number, there are even more. Let 1 ≤ ai ≤ 9 be a finite sequence of positive integers. Start with a positive integer n. In step i, replace the current number with the product of its digits times ai. Which starting numbers n can result in a sequence ending in n? For example, the number 28 can return to itself with multipliers {4,3,2}: 2×8×4 = 64, 6×4×3=72, 7×2×2=28. For a given n, what is the shortest and smallest sequence of ai which accomplish this?

Looked at another way, 9 is the largest number that can return using ai ≤ 1, 36 is the largest using ai ≤ 2, and 288 is the largest using ai ≤ 3. What is the largest number that has a sequence ai ≤ n?

What numbers can return using the same ai each time?

Problem #2:

Some numbers n have the property that if some occurrences of a particular digit string are replaced with the same digit, the resulting number is a factor of n. For example, for the substitution 30 → 7, we have the solutions 304130825 / 7417825 = 41, 430830 / 4787 = 90, 6303075 / 67775 = 93, and 1130304 / 11774 = 96. What are the small solutions? Which pairs can you prove have no solutions? What patterns are there?

Problem #3:

Given numbers n>10 and k>1, it is not hard to find a string of digits so that the number formed by every k consecutive digits is divisible by n. But only some pairs n and k have a unique longest sequence. For example, for n=17 and k=2, the only sequence of length 5 is 68517, since 68, 85, 51, and 17 are all multiples of 17. What are the longest unique solutions for small n and k?

Problem #4:

Let d(n) denote the number of divisors of n. What is the smallest solution to x+y=z if we are given the values of d(x), d(y), and d(z)?

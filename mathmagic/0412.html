Problem of the Month (April 2012)

Friedman numbers, numbers that can be written in a non-trivial way using their own digits using only +, –, ×, /, ^, concatenation of digits, and any number of parentheses, was the August 2000 Math Magic problem. There are several other web sites allowing differing operations: Vampire Numbers, Narcissistic Numbers, Wild Narcissistic Numbers, Pretty Wild Narcissistic Numbers. This month we investigate a related question with what we will generally call self-reproducing numbers. Which numbers can be written using only one or two operations? We allow !, √, and decimal points as well, since they are popular choices among recreational math enthusiasts. What about Fibonacci numbers, Catalan numbers, or binomial coefficients? What other operations yield self-reproducing numbers?

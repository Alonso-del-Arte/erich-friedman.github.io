Problem of the Month (September 2016)

On an N×N chessboard, when we place Q queens, what is the maximum number of squares that can be attacked exactly A times?

ANSWERS

George Sicherman sent solutions.

Let f(N,Q,A) denote the largest number of squares that are attacked exactly A times. Clearly f(N,Q,A) = 0 if Q<A or Q>N2–A.

The 2×2 boards are boring, as each square attacks each other. Thus f(2,Q,A) = N2–Q when Q=A, and 0 otherwise.

The boards containing one piece are also boring, as a queen on the edge of the board attacks the fewest squares, and a queen in the middle of the board attacks the most squares. Thus f(N,1,0) = N2–3N+2 and f(N,1,1) = N2–4N+4.

Computer evidence suggests the following formulas. Can you prove any of them? What other patterns can you find?

  • f(N,2,0) = N2–5N+7
  • f(N,2,1) = 7N–18 (for N≥6)
  • f(N,2,2) = 12 (for N≥8)

  • f(N,3,0) = N2–6N+9 (for N≥6)
  • f(N,3,3) = 5 (for N≥4)

  • f(N,4,0) = 4 (N–3)/2 (N–2)/2 (for N≥5)
  • f(N,4,2) = 6N–12 (for even N)
  • f(N,4,3) = 14 (for N≥6)
  • f(N,4,4) = 4 (for N≥4)

  • f(N,5,1) = 9N–31 (for N≥4)
  • f(N,5,4) = 8 (for N≥5)
  • f(N,5,5) = 3 (for N≥5)

  • f(N,6,5) = 4 (for N≥4)
  • f(N,6,6) = 2 (for N≥4)

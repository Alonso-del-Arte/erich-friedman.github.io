Line Segment Puzzles

In each rectangle, draw some line segments that pass through exactly 3 or 4 consecutive squares. The line segments can only be (H) horizontal, (V) vertical, or (D) diagonal. Every square must contain part of exactly one line segment. A square containing a letter contains part of a segment with that orientation. Each puzzle below has a unique solution.

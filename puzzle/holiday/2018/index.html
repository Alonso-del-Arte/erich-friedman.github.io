2. FULL HOUSE

Make the old year 2018 using only the digits 4, 4, 4, 6, 6. Then make the new year 2019 using only the digits 3, 3, 3, 6, 6. You can use addition, subtraction, multiplication, division, exponentiation, concatenation, and any number of parentheses.

3. STRANGE MATRIX

Find a table of distinct positive integers with 3 rows and 3 columns so that each row adds to 26 and each column multiplies to 240.

4. COMMON SIXES

Fill in letters to complete the most common 6-letter words in written English starting with each letter. The initial letters are given.
